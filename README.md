# Gendarmerie de Martinique - Connexion

Ce projet permet de créer une page de connexion sécurisée pour accéder à un site dédié à la Gendarmerie de Martinique. Le site utilise **HTML**, **CSS** et **JavaScript** pour une interface utilisateur propre et fonctionnelle.

## Fonctionnalités

- Page de connexion avec un identifiant et un mot de passe.
- Redirection vers un Google Sites ou une autre page après une connexion réussie.
- Design inspiré des couleurs du drapeau de la France.
- Menu déroulant pour certaines sections.

## Prérequis

- Un navigateur web moderne (Google Chrome, Mozilla Firefox, etc.).
- L'accès à Internet pour tester le déploiement en ligne.
